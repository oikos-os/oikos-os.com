# oikos-os.com

Landing page for [oikOS](https://github.com/oikos-os/oikOS) — the home for AI agents.

**Live:** [oikos-os.com](https://oikos-os.com)

## Stack

Static HTML + CSS. Hosted on GitHub Pages.

- Phosphor CRT aesthetic (amber on black)
- VT323 + IBM Plex Mono fonts
- Scanline + flicker + vignette effects
- No JavaScript frameworks. No build step.
